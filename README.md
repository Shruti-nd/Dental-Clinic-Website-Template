# 🦷 Dental Clinic Website Template

A modern, responsive dental clinic website template designed to help dental practices showcase services, build patient trust, and enable easy appointment booking.

This project is built using **HTML, Tailwind CSS, and JavaScript** and can be deployed instantly on platforms like **Vercel**.

---

# 🌐 Live Demo

After deployment your site will look like:

https://your-project-name.vercel.app

---

# ✨ Features

✔ Modern medical UI design
✔ Mobile responsive layout
✔ Services section with detailed descriptions
✔ Smile gallery (before & after)
✔ Patient testimonials
✔ Clinic photo gallery
✔ Appointment booking form
✔ WhatsApp consultation button
✔ Google Maps clinic location
✔ SEO-friendly structure
✔ Fast loading performance

---

# 🧰 Tech Stack

* HTML5
* Tailwind CSS
* JavaScript
* Swiper.js (image sliders)
* AOS.js (scroll animations)

---

# 📂 Project Structure

```
dental-clinic-website
│
├── index.html
├── README.md
├── images/
├── css/
└── js/
```

Images are stored locally inside the `images/` folder so the website works on any system.

---

# 🚀 Running Locally

Clone the repository:

```
git clone https://github.com/yourusername/dental-clinic-website.git
```

Open the project folder and simply run:

```
index.html
```

The website will open in your browser.

---

# ☁️ Deployment (Vercel)

1. Push project to GitHub
2. Login to Vercel
3. Import the repository
4. Click **Deploy**

Your site will be live in seconds.
